# Connection Ladder

A gamified web app helping people experiencing loneliness and social anxiety build social skills safely through a step-by-step format.

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **Lucide React** (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Routes

| Route | Description |
|-------|-------------|
| `/` | Welcome Page |
| `/auth/signup` | Sign Up Page |
| `/auth/login` | Log In Page |
| `/onboarding` | Onboarding Assessment |
| `/dashboard` | User Dashboard |
| `/challenge` | Active Challenge Page |

## Design System

The UI uses a minimalist, line-friendly design with natural, calming colors inspired by the [Connection Ladder UI Kit](https://connection-ladder-uikit.aura.build/).

- **Brand**: Warm neutrals (beige, taupe)
- **Accent**: Natural greens
- **Surface**: Soft off-whites

## Accessibility

- Semantic HTML (`<main>`, `<header>`, `<nav>`)
- Skip to main content link
- Visible focus states for keyboard navigation
- Reduced motion support
